Sentry2
=======

感谢选择 **Sentry2** 视觉传感器 :D

快速上手
------------------------

.. image:: images/sentry2_run_vision_by_stick.png

1. 将 Sentry2 通过 USB 连接至电脑
2. 上下拨动导航键即可快速切换算法
3. 将 Sentry2 摄像头面向需要识别的物体，即可看到识别情况
